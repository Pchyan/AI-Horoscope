需求是將星座運勢占卜軟體擴展為一個專業且多元的命理平台，包含多種運勢預測、個人化星盤、塔羅牌、八字、配對、專家諮詢等功能。這是一個大型產品設計，以下是功能拆解、模組規劃與未來開發建議，並會依照現有 React 架構給你可執行的擴展路線。

1. 功能模組化規劃
A. 星座運勢模組
支援每日、每週、每月、年度運勢（多檔期選擇）
多面向：健康、愛情、事業、財運
幸運數字、幸運色彩自動生成
個人化推播提醒
B. 個人星盤分析模組
使用者輸入出生年月日、時間、地點
自動計算並繪製本命盤（太陽、月亮、上升、行星、宮位）
解析行星落點、相位、宮位意義
行運盤、太陽返照盤分析
C. 塔羅牌占卜模組
多種牌陣（單張、三張、十字、愛情等）
牌義說明與圖示
支援自動解讀與心理建議
D. 命理工具整合模組
八字排盤、生辰八字分析
紫微斗數排盤
易經卜卦
手相、面相、姓名解析、夢境解讀
各模組可獨立查詢
E. 專業諮詢與配對模組
線上占星師/命理師即時諮詢（匿名提問/語音/視訊）
星座配對、戀愛/婚姻建議
兩人星盤合盤分析
F. 教育與知識庫模組
占星、命理、塔羅教學
每個結果頁皆可點擊展開知識說明
2. React 前端架構建議
多頁式路由（React Router）：每個模組一個主頁與子頁
元件化設計：每個占卜工具/分析結果為獨立元件
全局狀態管理（如 Redux 或 Context）：管理用戶資料、API 金鑰、推播設定
API 抽象化：所有運勢/命理查詢統一用 hooks 或 service 層
UI/UX：持續採用 Tailwind CSS，保持現代響應式設計
3. 優先擴展建議
第一階段（基礎命理功能）
運勢查詢支援「日/週/月/年」切換
增加幸運色、幸運數字顯示
星座配對查詢
塔羅牌簡易占卜
第二階段（個人化與命盤）
星盤輸入（年月日時地）與本命盤繪製
行運盤/太陽返照盤分析
八字/紫微/姓名/夢境等命理工具入口
第三階段（進階與專業化）
專家諮詢入口（可先做留言/匿名提問）
合盤分析/戀愛配對進階
教學與知識庫
4. 實作起點（以現有專案）
例：擴展「日/週/月/年」運勢查詢與幸運提醒
UI 增加「運勢類型」選單（每日、每週、每月、年度）
Gemini API prompt 增加查詢區間、幸運色/數字
結果區塊美化並顯示多欄位
例：星座配對初步設計
新增配對頁面，讓使用者選兩個星座
呼叫 Gemini API 生成配對分析
